# AutoDetail Pro

Сайт-визитка для автодетейлинга с формой заявок и админ-панелью.

## Деплой на Railway

1. Создай новый проект на [railway.app](https://railway.app)
2. Подключи GitHub репозиторий с этим кодом
3. В настройках переменных окружения добавь:
   - `ADMIN_PASSWORD` — пароль для входа в админку (по умолчанию `admin2024`)
   - `PORT` — Railway выставит автоматически

## Локальный запуск

```bash
npm install
npm start
```

Сайт будет доступен на http://localhost:8008

## Страницы

- `/` — главная страница сайта
- `/admin` — административная панель

## Вход в админку

- Логин: `admin`
- Пароль: значение переменной `ADMIN_PASSWORD` (по умолчанию `admin2024`)

## Стек

- Node.js + Express
- SQLite (база данных сохраняется в файл `autodetail.db`)
- Vanilla JS + CSS на фронте
